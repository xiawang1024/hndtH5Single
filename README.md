# h5-simple
基于parcel的简单页面（非spa）开发

## 特点
* 采用（rem+vw）方案适配手机端多分辨率
* css预处理器采用scss，内置常用mixin
* js兼容es6语法

## 建议（简单页面）：
* Dom操作库：zepto
* 请求操作库：axios
* 时间相关操作库：dayjs
* 视频播放器：video.js plyr.js
* CSS3动画相关：animate.css scrollreveal
* 全屏滚动：fullpage.js

## 快速开始

### 安装依赖
```
$ npm install
```
### 开发
```
$ npm run dev
```
### 打包
```
$ npm run build
```
